# Kintsugi Omakase

A premium cinematic sushi restaurant scroll experience built with Next.js 15, TypeScript, Tailwind CSS, GSAP ScrollTrigger, and Lenis.

## Run

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:3000`.

The supplied video is stored at `public/assets/omakase-dolly.mp4` and is scrubbed by the GSAP master scroll timeline.
